# Blogventure

A small React blog app I built while learning routing, forms, and API integration at NOVI Hogeschool.

## What's in here?
- Blog index with all posts
- Individual post pages with delete functionality
- Form to create new posts (with validation)
- Local JSON Server as the backend

## Tech stack
- React 18 + Vite
- React Router for navigation
- React Hook Form for form handling
- JSON Server for local API

## How to run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the API server (runs on port 3001):
   ```bash
   npm run server
   ```

3. In another terminal, start the app:
   ```bash
   npm run dev
   ```

4. Open http://localhost:5173
